# 🗺️ Система Поиска Маршрутов

## Обзор

Полная система для поиска маршрутов между городами Казахстана с админ панелью.

### 🎯 2 Основных Компонента

1. **RouteSearch.jsx** - Поиск маршрутов (для всех пользователей)
2. **AdminRoutes.jsx** - Управление маршрутами (только для администраторов)

---

## 📍 Список Городов

Система работает только с этими 15 городами Казахстана:

- Астана
- Алматы
- Шымкент
- Караганда
- Актобе
- Тараз
- Павлодар
- Усть-Каменогорск
- Семей
- Костанай
- Кызылорда
- Уральск
- Атырау
- Актау
- Балхаш

## 🔍 ЧАСТЬ 1: Поиск Маршрутов

### Страница: `/search-route`

#### Функционал:

- ✅ Выбор городов из dropdown списков
- ✅ Кнопка "Найти маршрут"
- ✅ Отображение расстояния и времени в пути
- ✅ Google Maps с маршрутом
- ✅ Умные остановки (для маршрутов > 500км)

#### Остановки:

Если маршрут дольше 500км, система автоматически добавляет 2-3 остановки:

- Показывает промежуточные города
- Расстояние от начала маршрута
- Кнопка "Подробнее" для каждой остановки

#### Пример использования:

```
1. Откуда: Астана
2. Куда: Алматы
3. Нажать "🔍 Найти маршрут"
4. Увидеть:
   - Расстояние: 1400 км
   - Время: 17 часов
   - Остановки: Караганда, Балхаш
   - Карта с маршрутом
```

## ⚙️ ЧАСТЬ 2: Админ Панель

### Страница: `/admin/routes`

#### Доступ:

Только для администраторов (требуется роль admin)

Таблица со всеми маршрутами:

| 1 | Астана | Алматы | 1400 км | 17 часов | 2 | ✏️ 🗑️ |

##### 3️⃣ **Редактировать маршрут**

- Нажать кнопку ✏️ в таблице

- Сохранить

##### 4️⃣ **Удалить маршрут**

- Нажать кнопку 🗑️ в таблице

## 💾 Логика Хранения

### localStorage

Все маршруты хранятся в **localStorage** браузера:

- Ключ: `routes_database`
- Структура: JSON массив маршрутов

### Структура Маршрута

```javascript
{
  id: 1234567890,                    // Уникальный ID (timestamp)
  from: "Астана",                    // Город отправления
  to: "Алматы",                      // Город назначения
  distance: 1400,                    // Расстояние в км
  duration: "17 часов",              // Время в пути
  description: "Маршрут...",         // Описание
  stops: [                           // Остановки
    {
      id: "stop_0",
      name: "Караганда",
      type: "остановка",
      distance: 700                  // км от начала
    },
    {
      id: "stop_1",
      name: "Балхаш",
      type: "остановка",
      distance: 1050
    }
  ],
  createdAt: "2026-04-02T..."       // Дата создания
}
```

## 📝 Пример Данных

### Предзаполненные маршруты:

Система не имеет предзаполненных маршрутов. Администратор добавляет их вручную.

### Автогенерируемые маршруты:

Если маршрут не найден в localStorage, система:

1. Вычисляет расстояние между городами
2. Рассчитывает время в пути (80 км/ч)
3. Генерирует остановки автоматически

## 🎯 Примеры Маршрутов

```javascript
// Маршрут Астана → Алматы
{
  from: "Астана",
  to: "Алматы",
  distance: 1400,
  duration: "17 часов",
  stops: [
    { name: "Караганда", type: "остановка", distance: 700 },
    { name: "Балхаш", type: "остановка", distance: 1050 }
  ]
}

// Маршрут Алматы → Шымкент
{
  from: "Алматы",
  to: "Шымкент",
  distance: 500,
  duration: "6 часов",
  stops: []  // Нет остановок (маршрут короче 500км)
}
```

## 🐛 Обработка Ошибок

### Валидация

- ✅ Города не могут быть пустыми
- ✅ Города не могут быть одинаковыми
- ✅ Маршрут не может быть дублирован
- ✅ Расстояние должно быть > 0

### Сообщения ошибок

- "❌ Выберите города"
- "❌ Выберите разные города"
- "❌ Этот маршрут уже существует"

## 📊 API сервиса routesService

```javascript
import routesService from "../services/routesService.js";

// Получить все маршруты
routesService.getRoutes();

// Поиск маршрута (существующего)
routesService.searchRoute(from, to);

// Найти/подгрузить маршрут и вернуть
routesService.findRoute(from, to);

// Добавить новый
routesService.addRoute({ from, to, distance, duration, description });

// Обновить
routesService.updateRoute(id, newData);

// Удалить
routesService.deleteRoute(id);

// Очистить все
routesService.clearAll();
```
