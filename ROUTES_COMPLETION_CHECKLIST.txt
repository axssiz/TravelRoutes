🚀 КАК НАЧАТЬ ИСПОЛЬЗОВАТЬ
═══════════════════════════════════════════════════════════════

1️⃣  ЗАПУСТИТЬ ПРИЛОЖЕНИЕ
   cd client
   npm start

2️⃣  ПЕРЕЙТИ НА СТРАНИЦУ ПОИСКА
   http://localhost:3000/search-route

3️⃣  АВТОРИЗОВАТЬСЯ (как администратор)
   Email: admin@example.com
   (или создать тестового админа)

4️⃣  ПЕРЕЙТИ НА АДМИН ПАНЕЛЬ
   http://localhost:3000/admin/routes

5️⃣  ДОБАВИТЬ МАРШРУТЫ
   Нажать "➕ Добавить маршрут"
   Заполнить форму
   Сохранить
