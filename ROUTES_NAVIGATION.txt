🔨 КОД - БЫСТРЫЙ СТАРТ

СКОПИРУЙ В КОНСОЛЬ БРАУЗЕРА (F12):

// ✅ Получить все маршруты
import routesService from './services/routesService.js';
routesService.getRoutes()

// ✅ Найти маршрут Астана → Алматы
routesService.findRoute('Астана', 'Алматы')

// ✅ Добавить маршрут
routesService.addRoute({
  from: 'Астана',
  to: 'Алматы',
  distance: 1400,
  duration: '17 часов',
  description: 'Маршрут через степь'
})

// ✅ Удалить маршрут по ID
const id = routesService.getRoutes()[0].id
routesService.deleteRoute(id)

  localStorage.removeItem('routes_database')

🔍 КАК РАБОТАЕТ СИСТЕМА

1. ПОИСК МАРШРУТА:
   Пользователь → Выбирает города → Система ищет в localStorage
   Если найден → Показывает данные + карта
   Если нет → Автогенерирует на основе координат

2. ОСТАНОВКИ:
   Если расстояние > 500км → система добавляет остановки
   Остановки = промежуточные города по пути

3. КАРТА:
   Google Maps API → Показывает маршрут между городами

⚠️  ВАЖНЫЕ ОГРАНИЧЕНИЯ

❌ НЕЛЬЗЯ:
   • Выбрать одинаковые города (Астана → Астана)
   • Использовать города не из списка
   • Добавить дублирующийся маршрут

✅ АВТОМАТИЧЕСКИ:
   • Рассчитывается расстояние
   • Рассчитывается время (80 км/ч)
   • Генерируются остановки
   • Преобразуются координаты
